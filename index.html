STEPS BAR

A steps progress bar is helpful when walking a user through a series of steps that are required to proceed.

For steps you need an unordered list, add a base class of steps-bar to your element and all increments will be handled by the counter on the element. There can be a max of how many steps and how wide your container is, for this you will want to be cautious and rethink this.

FYI -- CSS Counters can count and step for you.

For steps there is a top level class modifier of steps-bar--required which adds a disabled look and cursor to elements. This allows you to declare this once if its required then you will use JS to handle adding the state class of is-active to your active step.

ICONOGRAPHY

The Indominus Flex was built around the idea that adding icons to specific elements should be easy and try using as little markup as possible. Sure, you can still add the same old i tag but isn't it more fun to do cooler things. With that I have added a attribute called data-icon to specific elements that you add your ligature too, mainly used for material affects. This attribute in turn when added, adds all the neccesary rules for the ::before & ::after pseudo to use the icon with the font-family as well. So this should be a seemless process.
example: data-icon="delete"

From there you add your styling class and on the pseudo you prefer, add the attr(data-icon) to your content and bam that value is passed on and your icon will be displayed.

FLEXBOX GRID

Grid using the Flexible Box Layout Module.

The grid can be extended very easy and has modifiers for all the flexbox rules at all breakpoints. This includes align-item & align-self. Below is a list of all the class for flex and its items.

There is 2 levels for the grid with the base being flex & flex--12 added, then add flex__item--{*bp}-{*size} or flex__offset--{*bp}-{*size} to its children. The grid is focused on mobile up so xs is your base class for the grid.

Here is all modifier classes that are available.
Base: --{*bp}-flex-nowrap, --{*bp}-flex-wrap-rev, --{*bp}-horizontal-rev, --{*bp}-vertical, --{*bp}-vertical-rev
Alignment: --#{$bp}-start, --#{$bp}-center, --#{$bp}-end, --#{$bp}-around, --#{$bp}-between, --#{$bp}-top, --#{$bp}-middle, --#{$bp}-bottom
Item: flex__item--{*bp}-{*size}
